# Sunbla Khan - Extended CV Website

This repository contains my recruiter-focused personal website built with Jekyll and deployed on GitHub Pages.

Unlike a traditional one-page resume, this site is designed as an **extended CV**: a deeper, more human, and more practical view of who I am, what I build, and how I work.

## For Recruiters (30-Second Scan)

If you are reviewing my profile quickly, start here:

1. **Professional Summary + Impact:** `Home`
2. **Experience + Education + Skills Stack:** `Home`
3. **Project Evidence:** `Portfolio`
4. **Personality & Work Style:** `About` + `Personality`
5. **Direct Contact:** `Home` CTA or `Contact`

**Preferred Hiring Areas:** Data Science, AI Automation, Operations Analytics, and Process Optimization.

## Why This Website Exists

A regular CV is concise by design. It often cannot fully show:

- the context behind projects and results
- how I think and solve problems
- how I work with people
- my personality, values, and motivation
- what I am currently building and learning

This website gives recruiters and collaborators a clearer, more complete picture of my profile.

## What Recruiters Can Explore

- **Home:** high-level profile, impact metrics, experience, education, skills stack, testimonials, and direct contact CTA
- **About:** my story, growth journey, and personal values beyond job titles
- **Portfolio:** selected projects with technical and business outcomes
- **Personality:** work style, mindset, and human side of collaboration
- **Blog:** thoughts, learning notes, and project reflections
- **Contact:** direct ways to reach me (email, WhatsApp, LinkedIn, Upwork, Topmate)

## Website Goal

The goal is simple:

> Help recruiters quickly assess both my technical capability and my personal fit.

This site is meant to answer not only:
"Can she do the work?"
but also:
"How does she think, communicate, and grow?"

## Tech Stack

- Jekyll (static site)
- Markdown + custom HTML sections
- Custom CSS (dark professional theme)
- GitHub Pages (branch deploy: `main` / `root`)

## Deployment

This project is configured for GitHub Pages branch deployment.

1. Push to `sunblakhan.github.io`
2. Open repository **Settings -> Pages**
3. Select **Deploy from a branch**
4. Choose **main** and **/(root)**
5. Save and wait for the site to publish

## Contact Form Setup

The contact page uses Formspree placeholder:

`https://formspree.io/f/your-form-id`

Replace `your-form-id` with your real Formspree endpoint to receive form submissions.
