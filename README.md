# Sunbla Khan Portfolio (Jekyll)

This repository contains a complete Jekyll portfolio website ready for GitHub Pages branch deployment.

## Pages Setup

1. Push this project to your GitHub repository `sunblakhan.github.io`.
2. In repository settings, open **Pages**.
3. Set source to **Deploy from a branch**.
4. Select branch **main** and folder **/(root)**.
5. Save and wait for deployment.

## Main Sections

- Home
- About
- Skills
- Education
- Portfolio
- Personality
- Blog
- Contact

## Contact Form

The contact page uses Formspree placeholder:

`https://formspree.io/f/your-form-id`

Replace `your-form-id` with your real Formspree form endpoint.
